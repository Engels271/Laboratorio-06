# Objetivos:
- Crear una base de datos usando MongoDB Atlas.
- Diseñar y construir funciones en python que permitan realizar operaciones CRUD (crear, leer, actualizar y eliminar) sobre los documentos almacenados en una base de datos.
- Crear una página web que utilice frontend, backend y base de datos con Anvil.

# Resumen
El Laboratorio 6 consistió en configurar una base de datos MongoDB Atlas (clúster Tiendas y base de datos Tienda) con colecciones (empleados, productos, etc.) y luego desarrollar un script en Python (mongoDB.py) utilizando Pymongo para implementar las operaciones fundamentales CRUD, asegurando la validación y el manejo de errores en cada función. Posteriormente, se diseñó una aplicación web con el framework Anvil para una tienda, que incluía al menos dos formularios; el más importante permitía a los usuarios visualizar, filtrar y manipular (CRUD) los datos de la colección de productos, enlazando el frontend con la lógica Python de backend para gestionar la base de datos de forma dinámica y segura, culminando en la publicación del proyecto.
